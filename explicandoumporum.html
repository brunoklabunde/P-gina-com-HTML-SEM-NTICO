O que é a programação front-end?

A programação front-end é uma sequência de passos (códigos) que digitamos para gerar conteúdo em web sites, por exemplo. Essa programação é destinada a parte que fica visível ao usuário da aplicação, podendo utilizar html, css e javascript. O html, é a parte de inserir o conteúdo do site, o css é o que colore o site e define tamanhos, e o javascript é o que faz tudo funcionar.

Exemplos de linguagens de programação

Vamos explorar algumas dessas linguagens:

  • Javascript - Usado para adicionar interatividade a páginas da web.
  • Java - Usado para para desenvolver uma ampla gama de aplicações.
  • Php - Usado principalmente para criar páginas e aplicações web dinâmicas.
  • Python - Usado para desenvolvimento web, desenvolvimento de software, automação de tarefas, análise e ciência de dados, machine learning e inteligência artificial, e até mesmo para desenvolvimento de jogos e automação financeira.
  • C# - Uma linguagem de programação versátil, usada para desenvolver quase todos os tipos de aplicações.

Exemplos de Elementos Semânticos na linguagem de marcação html, não considerada de programação.

Vamos explorar alguns elementos semânticos:

  • header - Usado para introduzir cabeçalhos.
  • footer - Usado para informações de rodapé.
  • nav - Define uma área de navegação.
  • main - Envolve o conteúdo principal.
  • aside - Usado para informações complementares.
